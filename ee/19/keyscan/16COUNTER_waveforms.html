Sample behavioral waveforms for design file 16COUNTER.tdf

The following waveforms show the behavior of lpm_counter megafunction for the chosen set of parameters in design 16COUNTER.tdf. The design 16COUNTER.tdf is a 16 bit up counter. The counter supports a synchronous load of data input.

Fig. 1 : Wave showing counter operation.

The output port cout will be asserted at the completion of count sequence. The ports cin and cout are used to chain multiple counters together.
